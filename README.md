🚀 Click here to try: https://your-live-link.com

## 🚀 Features

- 🎯 Multiple-choice questions  
- ⏮️ Previous and Next navigation  
- ⏱️ Timer for the entire quiz  
- ✅ Saves user selected answers  
- 🔄 Allows changing answers before submission  
- 📊 Instant score calculation  
- 📄 Detailed answer sheet  
- ✔️ Shows correct answers  
- ❌ Shows wrong answers  
- ⚪ Displays unattempted questions  
- ⚡ Auto-submit when time ends
